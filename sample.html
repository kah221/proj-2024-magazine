Level 4 Heading (3rd Level Heading / Section)

italicな表示にするには、iタグを利用します。

boldな表示にするには、bタグを利用します。

Ctrl + Alt + Delのようなキーボード入力を指示する場合、kbdタグを利用します。ただし、+などはこのタグに含みません。

monospaceで表示するには、codeタグを利用するか、preとcodeタグを組み合わせてコードブロックにします。


<pre><code>
int main() {
    // このように
    return 0;
}
</code></pre>
  • 箇条書きを実装できます。
  • 箇条書きはul>liで実装できます。
    • liタグの中にさらにul>liを入れることで、ネストも可能です。
  1. 数字付き箇条書きを実装できます。
  2. 数字付き箇条書きはol>liで実装できます。
    1. liタグの中にさらにol>liを入れることで、ネストも可能です。

このプロジェクトは出版用途ですから、aタグはこのように利用できなくなります1。

1: 本来は<a>aタグ</a>と記述されていました。
